# Hackerrank Solutions Tracker

Проект для хранения и управления выполненными задачами с сайта HackerRank.

Цель: аккумулировать решения задач, их описания и код (с пояснениями) в одном репозитории, чтобы можно было повторно просматривать, улучшать и делиться ими.

**Ключевые идеи**

- Хранить каждое решение в отдельном файле с указанием языка, ссылки на задачу и краткого объяснения подхода.
- Сохранять как исходный код, так и возможные заметки / альтернативные решения.

## Запуск проекта и тестов

Этот репозиторий использует Vite + React + TypeScript. Основные команды:

```bash
npm install
npm run dev      # запуск в режиме разработки
npm run build    # сборка
npm run test     # запуск тестов (vitest / jest — смотрите package.json)
```
